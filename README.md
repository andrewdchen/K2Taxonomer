## K2Taxonomer

### Introduction

### Requirements

- R (>= 3.5)

### Installation

#### Install from GitHub

```r
devtools::install_github("montilab/K2Taxonomer")
```

#### Clone GitHub and install from source

```sh
## In console
git clone https://github.com/montilab/K2Taxonomer
```

```r
## In R
install.packages("path/to/K2Taxonomer", repos=NULL, type="source")
```
